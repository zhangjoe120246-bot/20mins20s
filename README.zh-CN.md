# 20min20s

基于 20-20-20 规则的 Windows 护眼提醒工具。

`20min20s` 只统计有效使用时长，不是单纯按开机或前台运行时间累加。如果一段时间内没有真实键盘或鼠标输入，计时会暂停；如果提醒触发时你正在全屏应用里，比如游戏、视频或演示，提醒会延后到退出全屏后再显示。

[English](./README.md)

## 功能

- 只统计有效使用时长，不把挂机时间算进去
- 支持按无输入时长自动暂停计时
- 支持全屏应用和指定前台进程下延后提醒
- 默认在有效使用 20 分钟后提醒远眺 20 秒
- 包含托盘状态、设置界面、统计视图和日志

## 下载

请从这里下载最新 Windows 发布包：

- [GitHub Releases](https://github.com/tiyadegure/20mins20s/releases/latest)

请使用完整的 zip 压缩包，不要只单独下载 `20min20s.exe`，因为程序运行依赖同目录下的 DLL 和其他运行时文件。

## 工作方式

- 计时只基于真实使用时长
- 一段时间没有键盘或鼠标输入时会自动暂停
- 全屏场景下触发的提醒会延后到退出全屏后再显示
- 托盘图标会反映当前状态，方便快速判断程序是否在计时、暂停、暂不提醒或等待补发提醒

## 使用

1. 下载最新 release 压缩包。
2. 解压到任意目录。
3. 运行 `20min20s.exe`。
4. 程序启动后会驻留在系统托盘。

## 说明

- 本项目仅支持 Windows。
- 当前正式维护和发布的仓库是 `20min20s`。
